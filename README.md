# Hookah Bar Scrollytelling Landing Page

Premium luxury hookah bar landing page with scroll-linked smoke animations.

## Tech Stack
- Next.js 14 (App Router)
- Framer Motion
- Tailwind CSS
- HTML5 Canvas

## Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. View at: `http://localhost:3000`
